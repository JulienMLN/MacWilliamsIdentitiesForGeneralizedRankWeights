# MacWilliamsIdentitiesForGeneralizedRankWeights

This repository contains a Jupyter notebook with all the codes, written in SAGE, related to the paper MacWilliams identites for the generalized rank weights that can be found on Arxiv : 

1. Download the Jupyter Notebook MacWilliamsGRW_Codes_ipynb
2. Run it with a SageMath kernel in order to get all the functionalities work.
3. Test the codes.
